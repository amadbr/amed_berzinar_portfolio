## Features

- Responsive design optimized for all devices
- Dark / light theme toggle with smooth transitions
- Scroll-triggered animations powered by Framer Motion
- Project showcase with detailed case studies and image galleries
- Static site generation (SSG) for fast page loads and SEO
- Deployed as a static export on Netlify

## Project Structure

```
src/
├── app/                  # Next.js App Router pages
│   ├── page.tsx          # Home page
│   └── projects/[slug]/  # Dynamic project detail pages
├── components/           # Reusable UI components
│   ├── Navbar.tsx        # Navigation with mobile menu
│   ├── Hero.tsx          # Landing section
│   ├── About.tsx         # About section
│   ├── Skills.tsx        # Tech stack display
│   ├── Projects.tsx      # Project cards grid
│   ├── ProjectDetail.tsx # Project case study page
│   └── Contact.tsx       # Contact form
└── data/
    └── portfolio.ts      # All portfolio data & project content
```

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
